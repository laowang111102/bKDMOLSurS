# 花卉识别分类系统 python170

## 项目概述

花卉识别分类系统是一款基于深度学习的图像分类应用，使用 Python 编写，采用 PyTorch 框架下的 ResNet18 网络进行训练。该系统作为深度学习课程的作业案例，旨在提供机器学习指导，同时具备一定的通用性，适用于其他分类问题的训练。

## 技术栈

- 编程语言：Python
- 深度学习框架：PyTorch
- 网络结构：ResNet18

## 功能模块

- 数据集处理
- 模型训练
- 模型评估
- 图形用户界面（GUI）

## 系统角色

- 用户：使用系统进行花卉图像的识别与分类
- 开发者：进行系统的开发、训练与维护

## 运行环境

- 操作系统：支持 Windows/Linux/OS X
- Python 版本：3.6 或以上
- PyTorch 版本：根据具体环境进行适配
- 依赖库：详见表依赖列表

## 部署步骤

1. 安装 Python 环境
2. 安装 PyTorch 框架
3. 下载预训练权重与数据集
4. 运行 GUI 程序
